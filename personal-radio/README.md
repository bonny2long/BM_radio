# BM Radio

Private premium radio, music player, and audiobook bookshelf built from NAS media library.

## Project Structure

- `backend/`: FastAPI backend service.
- `frontend/`: React + Vite + TypeScript frontend service.
- `scripts/`: Maintenance and scanning scripts.

## Getting Started

1. Copy `.env.example` to `.env` and adjust paths.
2. Setup backend environment and install requirements.
3. Setup frontend and install npm packages.
